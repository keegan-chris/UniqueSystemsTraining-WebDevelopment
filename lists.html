HTML Lists

There are 3 types of lists in HTML


Ordered List or Numbered List
UnOrdered List or Bullet List
Description List or Definition List
Example of Ordered List:
  1. Apple
  2. Banana
  3. Strawberry
  4. WaterMelon
  5. Kiwi
Example of Unordered List:
  • Toothbrush
  • Face Wash
  • Face Cream
  • Deodorant
  • Soap
Example of Definiton List:
Computer
Electronic device for storing and calculating
Program
StepWise instructions given to solve any problem
